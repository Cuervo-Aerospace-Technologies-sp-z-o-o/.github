## Cuervo Aerospace Technologies

We build software that helps small aviation maintenance organisations go paperless.

**[cuervo.aero](https://cuervo.aero)** is a cloud platform for managing maintenance documentation in EASA-regulated environments — work packages, job cards, digital signatures and audit trail, all in one system.

### What we solve

Small Part-145 MROs, CAMO offices and GA workshops still run on printed task cards, wet-ink signatures and binders shipped by post. Inspectors ask for records, and someone spends a day photocopying. cuervo.aero replaces that workflow with a single digital system — from work order creation to signed PDF package.

### Built for

- **Part-145 MROs** — digital work orders, dual-signature job cards (MECH + INSP), automatic PDF generation
- **CAMO / Part-CAMO** — audit-ready documentation, fleet registry, compliance tracking
- **Aircraft Mechanics (Part-66)** — mobile-first interface for the hangar, sign tasks on your phone
- **Quality Managers** — full audit trail, occurrence reporting (EU 2015/1018), inspection readiness

### Stack

Symfony &middot; Next.js &middot; PostgreSQL &middot; Redis &middot; Docker &middot; S3-compatible storage

### Compliance

EASA Part-M &middot; Part-ML &middot; Part-CAMO &middot; Part-145 &middot; EU 2015/1018
